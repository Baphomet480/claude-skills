# LinkedIn Engagement Session Checklist

Run through this checklist for each LinkedIn session. Not every step applies every time.

## Pre-Flight

- [ ] Chrome browser automation connected
- [ ] Signed into LinkedIn in Chrome profile
- [ ] Read campaign memory file for context on recent activity

## Review Phase

- [ ] Navigate to activity page
- [ ] Scan all posts for new comments since last session
- [ ] Identify comments worth replying to (prioritize by):
  1. **High-value connections** (potential clients, referral sources, industry leaders)
  2. **Substantive comments** that invite technical discussion
  3. **Questions** from anyone (always reply to questions)
  4. Generic praise (like or short thanks, no detailed reply needed)

## Reply Phase

- [ ] Draft reply for each priority comment
- [ ] Get user approval on each draft before posting
- [ ] Post replies one at a time, confirming each was posted
- [ ] Screenshot or verify each reply is live

## Content Phase (if creating a new post)

- [ ] Check untapped content pipeline (blog posts, war stories, campaign notes)
- [ ] Draft post following Content Guidelines
- [ ] Review for AI tells (triple repetitions, balanced closers, parallel structures)
- [ ] Get user approval
- [ ] Publish or schedule
- [ ] Add blog link as first comment (if publishing now)
- [ ] If scheduling: create calendar reminder to add first comment when post goes live

## Post-Session

- [ ] Update campaign memory with:
  - Comments replied to (who, which post, topic)
  - Posts published or scheduled (title, date)
  - Notable engagement changes
  - Next post ideas discussed
