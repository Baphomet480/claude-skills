# Style Word Bank

Use these art-direction keywords with the Redesign mode (Nano Banana Pro) for specific visual transformations. Combine terms from different categories.

## Layout & Structure

| Keyword | Effect |
|---|---|
| **Bento Grid** | Modular, boxy, card-based layouts. Distinct compartments in a cohesive grid hierarchy. |
| **Editorial** | High-fashion magazine feel. Large serif headings, generous whitespace, asymmetrical images. |
| **Swiss Style** | Objectively clear. Grid systems, sans-serif typography, flush-left text. |
| **Split-Screen** | Distinct vertical division pairing a solid color block with full-bleed imagery. |

## Texture & Depth

| Keyword | Effect |
|---|---|
| **Glassmorphism** | Translucency, background blur (backdrop-filter), subtle white borders. "Frosted glass." |
| **Claymorphism** | Soft, inflated 3D shapes with inner shadows. Friendly, approachable, tactile. |
| **Skeuomorphic** | Realistic textures (leather, paper, metal) and controls that look like physical switches. |
| **Grainy/Noise** | Film grain or texture overlays on gradients to reduce "digital" shine and add warmth. |

## Atmosphere & Era

| Keyword | Effect |
|---|---|
| **Brutalist** | Raw, unpolished, default system fonts, high contrast, hard edges. "Ugly-cool." |
| **Cyberpunk** | Dark mode, neon accents (cyan/magenta), glitch effects, tech-heavy interfaces. |
| **Y2K** | Late 90s/Early 2000s optimism. Chrome textures, bubble letters, bright blues and pinks. |
| **Retro-Futurism** | 80s Synthwave. Sunsets, wireframe grids, VHS aesthetics, glowing lines. |

## Color & Contrast

| Keyword | Effect |
|---|---|
| **Duotone** | Entire UI composed of two contrasting colors (and their shades). |
| **Monochromatic** | Single base hue for a cohesive, branded look. |
| **Pastel Goth** | Soft, milky pastel colors paired with stark black typography and borders. |
| **Dark Mode OLED** | True black backgrounds (#000000) rather than dark grey, high contrast and pop. |

## Combining Keywords

Effective prompts combine 2-3 keywords across categories:

- "Redesign using **Bento Grid** layout with **Glassmorphism** cards and **Dark Mode OLED** background"
- "Apply **Editorial** layout with **Grainy** textures and **Duotone** color scheme in sage green"
- "Transform to **Brutalist** style with **Split-Screen** layout and **Monochromatic** palette"
- "Redesign as **Y2K** aesthetic with **Claymorphism** buttons and bright pink **Duotone**"
