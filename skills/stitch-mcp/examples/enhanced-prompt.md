# Example: Prompt Enhancement

## User Input (Vague)
> "Make a landing page for my fitness app called 'Pulse'. Use red and black. It should have a place for features and a hero section."

## Enhanced Prompt (Stitch-Optimized)

A high-energy, modern landing page for "Pulse", a fitness tracking application. The vibe is powerful, motivating, and high-performance.

**PAGE STRUCTURE:**
1. **Header:** Minimalist navigation with Pulse logo (left) and "Start Training" primary CTA button (right). Sticky on scroll.
2. **Hero Section:** Full-width dark background with large, emotive fitness photography. Headline: "Elevate Every Beat." Subheadline: "Track, analyze, and crush your fitness goals with Pulse." Primary red CTA: "Get Started Free".
3. **Feature Grid:** Three-column card layout highlighting:
   - **Real-time Tracking**: Live stats from your wearable device
   - **AI Coaching**: Personalized workouts based on performance data
   - **Community Challenges**: Compete with friends and stay motivated
4. **Social Proof Section:** Subtle horizontal slider showing "Trusted by 500,000+ athletes" with partner logos.
5. **Footer:** Quick links (Training, Pricing, Support), social icons, and legal text.

## Design System (if not attached to project)

When no design system is attached, include this block:

**DESIGN SYSTEM:**
- Platform: Web, Desktop-first
- Theme: Dark
- Primary: Racing Red (#e11d48) for CTAs and active states
- Background: Deep Obsidian (#0f172a)
- Surface: Slate (#1e293b) for cards
- Typography: Bold condensed sans-serif headings, clean body text
- Style: ROUND_EIGHT corners, minimal shadows, high contrast

## With a Design System Attached

When a design system is already attached to the project via `create_design_system`, you can omit the design system block entirely. The prompt becomes just:

A high-energy landing page for "Pulse", a fitness tracking application. Powerful and motivating.

**PAGE STRUCTURE:**
1. **Header:** Minimalist nav with Pulse logo (left) and "Start Training" CTA (right). Sticky.
2. **Hero Section:** Full-width dark hero with fitness photography. Headline: "Elevate Every Beat." CTA: "Get Started Free".
3. **Feature Grid:** Three cards: Real-time Tracking, AI Coaching, Community Challenges.
4. **Social Proof:** "Trusted by 500,000+ athletes" with partner logos.
5. **Footer:** Quick links, social icons, legal.

The design system handles colors, fonts, and roundness automatically.
